# Active Flow Sculpting — Characterization

Quick overview of the Characterization toolkit for Active Flow Sculpting experiments.

## About

This repository provides the Characterization toolkit for Active Flow Sculpting experiments. It offers a Streamlit-based interactive application to compute Mixing Coefficient, Moment of Inertia (MoI), Fluid Dislocation/Displacement, and Uniformity analyses from microscope-derived particle images exported by the ActiSculpt pipeline. The app supports batch processing, interactive parameter tuning, and exports results in JSON and image formats for downstream analysis.

## How to use the app

- Double-click the appropriate launcher for your OS (`run_app.bat` on Windows, `run_app.command` or `run_app.sh` on macOS/Linux), or run the app manually:

```bash
streamlit run GUI.py
```

- In the sidebar select your dataset JSON and the folder containing the image files.
- Choose the analysis type (Mixing, Uniformity, MoI, Fluid Dislocation), set amplitudes and other parameters, and run the analysis.
- Use the interactive plots and summaries to inspect results, then export consolidated `Results.json` and any per-particle cropped images as needed.

What it does: A Streamlit-based interactive application to compute Mixing Coefficient,
Moment of Inertia (MoI), Fluid Dislocation / Displacement, and Uniformity analyses from
microscope-derived particle images exported by the ActiSculpt pipeline.

## Features
- Mixing coefficient, Uniformity (MoI) and Fluid Displacement calculations
- Batch processing across experiment folders generated by the acquisition pipeline
- Image cropping, channel separation, and per-frame binarization utilities
- JSON-backed results and optional exports for downstream plotting and archiving

## Entry point
- The main app is `GUI.py` — run with Streamlit: `streamlit run GUI.py`.

## Cross-platform launchers
For convenience there are one-click launchers for each OS in this folder:
- macOS / Linux: `run_app.command` and `run_app.sh` (double-click or run in Terminal)
- Windows: `run_app.bat` (double-click to start)

These scripts will create and use a local virtual environment at `.venv/` if one
doesn't already exist, install `requirements.txt` if present, and launch the Streamlit UI.

## Minimum requirements
- Python 3.9+ recommended
- Streamlit and standard scientific Python stack (listed in `requirements.txt` if present)

## Usage
1. Double-click the platform-appropriate launcher or run `streamlit run GUI.py` from a Python environment with dependencies installed.
2. Use the sidebar controls to point the app at your dataset JSON and image folders.
3. Choose analysis type (Mixing, Uniformity, MOI, Fluid Dislocation), set amplitudes, and run.
4. Export or inspect aggregated results from the in-app summary and plots.

## Troubleshooting
- If the launcher fails with "permission denied" on macOS, make the script executable: `chmod +x run_app.command`.
- If Python or Streamlit are not found, activate the virtual environment manually:
  ```bash
  source .venv/bin/activate
  pip install -r requirements.txt
  streamlit run GUI.py
  ```
